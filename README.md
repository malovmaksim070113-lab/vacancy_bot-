# Telegram Vacancy Bot

## Что делает
- Парсит вакансии «Системный аналитик»
- Фильтрует по зарплате
- Публикует вакансии в Telegram-канал

## Как запустить
1. Залить проект в GitHub
2. Задеплоить в Railway
3. В Railway → Variables добавить:
   BOT_TOKEN=токен_бота

## Команды
/start — запуск  
/check — ручная проверка
